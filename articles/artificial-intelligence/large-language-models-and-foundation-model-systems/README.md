# Large Language Models and Foundation Model Systems

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Large Language Models and Foundation Model Systems

It includes:

- Python workflow for LLM system evaluation, grounding review, safety/security scoring, cost analysis, and governance summaries.
- R workflow for LLM evaluation summaries by use case and risk level.
- SQL schema for model records, prompt versions, retrieval logs, tool-call logs, evaluation results, safety reviews, incident records, and governance review.
- Rust CLI for validating LLM evaluation records and risk thresholds.
- Go monitoring API for LLM foundation-model system metrics.
- Julia workflow for token-cost, latency, and system-risk simulation.
- TypeScript dashboard logic for LLM system governance review.
- C++ example for attention-style scoring and token-budget estimation.
- Advanced notebooks for prompt evaluation, retrieval grounding, cost/latency review, prompt-injection testing, and governance registers.
- Documentation templates for LLM system cards, prompt review, retrieval evaluation, tool-use approval, and incident response.
