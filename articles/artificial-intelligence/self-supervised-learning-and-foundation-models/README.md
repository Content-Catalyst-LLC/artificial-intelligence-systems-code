# Self-Supervised Learning and Foundation Models

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Self-Supervised Learning and Foundation Models

It includes:

- Python workflow for self-supervised pretraining review, foundation-model utility scoring, data risk scoring, and governance summaries.
- R workflow for objective-level and modality-level foundation model evaluation.
- SQL schema for pretraining runs, datasets, model objectives, evaluation results, adaptation records, risk reviews, monitoring, and governance artifacts.
- Rust CLI for validating foundation-model metadata and pretraining governance thresholds.
- Go monitoring API for foundation-model governance metrics.
- Julia workflow for self-supervised learning curve and representation utility simulation.
- TypeScript dashboard logic for foundation-model risk and governance review.
- C++ example for contrastive similarity scoring and representation evaluation.
- Advanced notebooks for masked prediction, contrastive learning, reconstruction, foundation-model risk, and governance registers.
- Documentation templates for foundation model cards, pretraining data records, self-supervised objective review, and deployment approval.
