# Adversarial Resilience Deployment Readiness Checklist

## Threat Modeling

- [ ] Protected assets documented.
- [ ] Attack surfaces documented.
- [ ] Attacker capabilities documented.
- [ ] High-risk workflows identified.
- [ ] Threat model reviewed.

## Testing

- [ ] Clean performance evaluated.
- [ ] Corruption robustness evaluated.
- [ ] Adversarial robustness evaluated.
- [ ] OOD behavior evaluated.
- [ ] Prompt-injection tests completed.
- [ ] Retrieval-poisoning tests completed.
- [ ] Tool-abuse tests completed.
- [ ] Privacy leakage tests completed.

## Controls

- [ ] Input validation implemented.
- [ ] Monitoring signals active.
- [ ] Human review gates configured.
- [ ] Least-privilege permissions configured.
- [ ] Sandbox controls tested.
- [ ] Rollback path tested.
- [ ] Incident response owner assigned.
