# Robustness and Adversarial Resilience in Machine Learning

This article folder supports the article "Robustness and Adversarial Resilience in Machine Learning" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- clean accuracy and robust accuracy
- adversarial-style perturbation experiments
- perturbation-budget stress tests
- robustness gap diagnostics
- distribution-shift and corruption reporting
- threat-model documentation
- SQL schemas for robustness evaluation metadata
- resilience governance checklists
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
