# Model Card Notes for Computer Vision and Machine Perception

Recommended documentation fields:

- model family
- visual task type
- training dataset sources
- label method
- image domains represented
- sensor and camera assumptions
- known lighting and viewpoint limitations
- intended use
- out-of-scope use
- accuracy and error metrics by group or condition
- IoU or detection metrics where relevant
- known failure modes
- adversarial and robustness testing
- privacy and retention rules
- human oversight process
- correction and appeal mechanisms
