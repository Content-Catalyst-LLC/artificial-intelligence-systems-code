# Computer Vision and Machine Perception

This article folder supports the article "Computer Vision and Machine Perception" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- synthetic image generation
- convolution and edge detection
- feature extraction concepts
- CNN representation intuition
- vision transformer patch embeddings
- IoU and segmentation metrics
- robustness and perturbation tests
- grouped computer vision error diagnostics
- SQL metadata for vision AI systems
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
