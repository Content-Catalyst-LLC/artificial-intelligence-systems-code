# Planning, Search, and Sequential Decision Systems

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Planning, Search, and Sequential Decision Systems

It includes:

- Python workflow for A* search, planning risk, uncertainty review, reversibility review, and governance summaries.
- R workflow for sequential decision evaluation summaries.
- SQL schema for planning systems, state definitions, action registries, plans, policy evaluations, search traces, human approvals, incidents, rollback, and governance reviews.
- Rust CLI for validating planning records and action-risk thresholds.
- Go monitoring API for planning and sequential decision metrics.
- Julia workflow for value iteration and sequential decision risk simulation.
- TypeScript dashboard logic for planning governance review.
- C++ example for planning-risk scoring and action routing.
- Advanced notebooks for state-space search, value iteration, policy evaluation, agent planning traces, constraint review, and governance registers.
- Documentation templates for planning system cards, action registry reviews, reward reviews, policy evaluation, human approval gates, rollback readiness, incident response, and deployment readiness.
