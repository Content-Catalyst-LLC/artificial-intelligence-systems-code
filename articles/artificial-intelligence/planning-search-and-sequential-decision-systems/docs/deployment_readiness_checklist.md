# Planning System Deployment Readiness Checklist

## Representation

- [ ] State variables documented.
- [ ] Known omitted variables documented.
- [ ] Partial observability assumptions documented.
- [ ] Transition assumptions documented.

## Action Governance

- [ ] Action registry completed.
- [ ] High-impact actions identified.
- [ ] Irreversible actions identified.
- [ ] Human approval gates configured.
- [ ] Permission boundaries tested.

## Evaluation

- [ ] Search or policy evaluation completed.
- [ ] Constraint tests completed.
- [ ] Simulation validation completed.
- [ ] Failure cases reviewed.
- [ ] Monitoring signals defined.

## Operations

- [ ] Trace logging active.
- [ ] Rollback route tested.
- [ ] Incident response owner assigned.
- [ ] Governance review scheduled.
