# AI for Scientific Discovery and Computational Research

This article folder supports the Artificial Intelligence Systems knowledge-series article:

AI for Scientific Discovery and Computational Research

It includes:

- Python workflow for active learning, surrogate modeling, candidate ranking, and discovery governance.
- R workflow for reproducibility diagnostics and scientific model review.
- SQL schema for scientific projects, datasets, experiments, simulations, model runs, candidates, hypotheses, validation, and reproducibility.
- Rust CLI for validating candidate records and scientific metadata.
- Go monitoring API for scientific workflow status and model-run summaries.
- Julia workflow for scientific surrogate simulation and discovery optimization.
- TypeScript dashboard logic for computational research review.
- C example for low-level numerical kernel validation.
- C++ example for candidate ranking and acquisition scoring.
- Fortran example for grid-based scientific simulation.
- Advanced notebook generator for representation learning, active learning, surrogate models, reproducibility, and governance.
- Documentation templates for AI-assisted scientific discovery governance, reproducibility review, and computational research metadata.
