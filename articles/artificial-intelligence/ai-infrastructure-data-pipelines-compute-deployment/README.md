# AI Infrastructure: Data Pipelines, Compute, and Deployment Systems

This article folder supports the article "AI Infrastructure: Data Pipelines, Compute, and Deployment Systems" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- pipeline DAG modeling
- compute utilization diagnostics
- serving capacity analysis
- latency-budget modeling
- infrastructure readiness scoring
- MLOps risk scoring
- observability metadata
- SQL schemas for AI infrastructure governance
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
