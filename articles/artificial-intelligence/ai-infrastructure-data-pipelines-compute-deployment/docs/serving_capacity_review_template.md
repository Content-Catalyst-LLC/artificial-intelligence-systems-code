# AI Serving Capacity Review Template

Recommended fields:

- service name
- model version
- deployment environment
- expected request volume
- required QPS
- throughput per replica
- required replicas
- latency budget
- observed p50 latency
- observed p95 latency
- observed p99 latency
- feature lookup latency
- model inference latency
- network latency
- rollback strategy
- autoscaling policy
- monitoring owner
- incident response owner
