# Edge AI and Distributed Intelligence

This article folder supports the article "Edge AI and Distributed Intelligence" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- edge-node simulation
- cloud versus edge latency analysis
- bandwidth savings analysis
- federated averaging
- resource-constrained inference diagnostics
- node trust and governance scoring
- distributed intelligence risk analysis
- SQL schemas for edge AI governance
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
