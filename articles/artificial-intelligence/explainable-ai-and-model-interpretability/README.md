# Explainable AI and Model Interpretability

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Explainable AI and Model Interpretability

It includes:

- Python workflows for local explanations, surrogate models, counterfactuals, feature importance, and stability tests.
- R workflows for transparent baselines, local contribution analysis, and explanation audits.
- SQL schema for explanation governance, model cards, explanation logs, counterfactual records, and audit evidence.
- Rust CLI for validating explanation artifacts.
- Go API for serving explanation audit summaries.
- Julia workflow for attribution stability and counterfactual distance simulation.
- TypeScript dashboard logic for explanation review and governance interfaces.
- Documentation templates for explanation audits, model cards, and counterfactual review.
- Notebook generator for advanced interpretability notebooks.
