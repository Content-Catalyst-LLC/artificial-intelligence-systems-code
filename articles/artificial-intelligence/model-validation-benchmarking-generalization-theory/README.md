# Model Validation, Benchmarking, and Generalization Theory

This article folder supports the article "Model Validation, Benchmarking, and Generalization Theory" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- train, validation, and test split diagnostics
- empirical risk and generalization-gap analysis
- cross-validation workflows
- calibration and expected calibration error
- distribution-shift simulation
- benchmark-saturation scoring
- model evaluation metadata
- SQL schemas for validation governance
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
