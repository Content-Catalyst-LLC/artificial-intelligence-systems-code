# Model Validation Governance Checklist

Recommended governance questions:

1. What is the intended use of the model?
2. What decisions will model outputs influence?
3. Are train, validation, and test sets separated correctly?
4. Was the test set used only for final evaluation?
5. Are data leakage risks documented?
6. Are metrics aligned with decision costs?
7. Are subgroup results reported?
8. Is calibration measured?
9. Is uncertainty reliable under shift?
10. Has external validation been performed?
11. Has distribution shift been tested?
12. Are benchmark limitations documented?
13. Is benchmark saturation a concern?
14. Are known failure modes recorded?
15. Is post-deployment monitoring defined?
