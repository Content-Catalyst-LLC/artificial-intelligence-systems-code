# Synthetic Evaluation Deployment Readiness Checklist

## Synthetic Data

- [ ] Intended use documented.
- [ ] Source data provenance documented.
- [ ] Generation method documented.
- [ ] Fidelity metrics reviewed.
- [ ] Utility metrics reviewed.
- [ ] Privacy risk reviewed.
- [ ] Rare-case coverage reviewed.
- [ ] Fairness and subgroup coverage reviewed.

## Simulation

- [ ] Scenario assumptions documented.
- [ ] Sim-to-real validation completed.
- [ ] Domain-expert review completed.
- [ ] Failure modes documented.
- [ ] Transfer limits documented.

## Benchmark and Governance

- [ ] Benchmark contamination risk reviewed.
- [ ] Metrics and scoring rules documented.
- [ ] Approved claims documented.
- [ ] Invalid claims documented.
- [ ] Review cadence scheduled.
