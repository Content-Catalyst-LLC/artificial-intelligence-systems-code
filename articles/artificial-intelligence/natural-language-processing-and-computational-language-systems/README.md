# Natural Language Processing and Computational Language Systems

This article folder supports the article "Natural Language Processing and Computational Language Systems" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- tokenization
- n-gram language modeling
- embedding similarity
- retrieval simulation
- attention intuition
- perplexity and language model evaluation
- text classification diagnostics
- hallucination-risk logging concepts
- SQL metadata for computational language systems
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
