# Clinical AI Deployment Readiness Checklist

## Intended Use

- [ ] Intended use documented.
- [ ] Prohibited use documented.
- [ ] Clinical owner assigned.
- [ ] Technical owner assigned.
- [ ] Human oversight policy documented.
- [ ] Regulatory status reviewed.

## Validation

- [ ] Local validation completed.
- [ ] Calibration reviewed.
- [ ] Threshold policy reviewed.
- [ ] Subgroup performance reviewed.
- [ ] Workflow impact reviewed.
- [ ] Alert burden reviewed.
- [ ] Safety review completed.

## Privacy and Security

- [ ] Data sources approved.
- [ ] Access controls configured.
- [ ] Audit logging active.
- [ ] Vendor review completed.
- [ ] Incident response path defined.

## Operations

- [ ] Monitoring signals active.
- [ ] Change-control process defined.
- [ ] Rollback path tested.
- [ ] Clinician training completed.
- [ ] Governance review scheduled.
