# Trust, Interpretability, and User-Centered AI Systems

This article folder supports the article "Trust, Interpretability, and User-Centered AI Systems" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- trust calibration
- model confidence diagnostics
- explanation-quality scoring
- user reliance simulation
- overreliance and underreliance analysis
- human-AI interaction metadata
- SQL schemas for interaction logs and review workflows
- design-review checklists
- model-card notes
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
