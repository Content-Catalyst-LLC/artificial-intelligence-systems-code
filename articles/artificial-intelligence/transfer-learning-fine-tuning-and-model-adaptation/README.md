# Transfer Learning, Fine-Tuning, and Model Adaptation

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Transfer Learning, Fine-Tuning, and Model Adaptation

It includes:

- Python workflow for adaptation experiment simulation, transfer gain, forgetting risk, and governance scoring.
- R workflow for method-level transfer learning experiment review.
- SQL schema for base models, adapted models, fine-tuning runs, adapters, LoRA records, evaluation results, deployment approvals, drift monitoring, and governance review.
- Rust CLI for validating adaptation experiment metadata and deployment thresholds.
- Go monitoring API for model adaptation governance metrics.
- Julia workflow for adaptation loss and forgetting-risk simulation.
- TypeScript dashboard logic for fine-tuning experiment review.
- C++ example for low-rank adaptation and transfer-gain scoring.
- Advanced notebooks for fine-tuning tradeoffs, negative transfer, parameter-efficient adaptation, and adaptation governance.
- Documentation templates for adapted model cards, fine-tuning review, adapter registry, and deployment approval.
