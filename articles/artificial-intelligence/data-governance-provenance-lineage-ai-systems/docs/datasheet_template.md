# Dataset Datasheet Template

Recommended fields:

- dataset name
- dataset version
- dataset owner
- motivation and intended use
- prohibited or inappropriate uses
- source systems
- collection method
- collection period
- population represented
- known exclusions
- preprocessing and transformations
- label sources
- quality checks
- bias and representation review
- privacy and sensitivity review
- licensing and rights
- retention policy
- maintenance plan
- governance owner
