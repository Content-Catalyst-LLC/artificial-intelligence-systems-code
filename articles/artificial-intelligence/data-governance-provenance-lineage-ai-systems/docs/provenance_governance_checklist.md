# Data Governance, Provenance, and Lineage Checklist

Recommended governance questions:

1. What source systems produced the dataset?
2. Who or what generated the data?
3. What transformations were applied?
4. Which code version produced each derived artifact?
5. Which datasets produced each feature table?
6. Which feature table produced each model?
7. Which evaluation report applies to each model version?
8. Which deployment used each model version?
9. Are dataset rights and licenses documented?
10. Are sensitive fields identified and controlled?
11. Are data quality checks recorded?
12. Are documentation artifacts complete?
13. Can the model be reproduced from data, code, environment, and hyperparameters?
14. Can source-data defects be traced to affected models?
15. Is governance review embedded in the lifecycle?
