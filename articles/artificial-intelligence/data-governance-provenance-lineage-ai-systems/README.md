# Data Governance, Provenance, and Lineage in AI Systems

This article folder supports the article "Data Governance, Provenance, and Lineage in AI Systems" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- provenance graph modeling
- W3C PROV-inspired entity/activity/agent metadata
- data lineage and transformation tracing
- impact analysis for source data defects
- data-quality review metadata
- governance documentation templates
- SQL schemas for AI data governance
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
