# Agent Deployment Readiness Checklist

## Architecture

- [ ] Agent purpose is documented.
- [ ] Tool registry is reviewed.
- [ ] Tool permissions follow least privilege.
- [ ] State and memory retention are documented.
- [ ] High-risk tools require confirmation.
- [ ] Sensitive actions require human approval.
- [ ] Sandbox controls are tested.
- [ ] Rollback procedure is documented.

## Evaluation

- [ ] Task-success tests completed.
- [ ] Tool-selection tests completed.
- [ ] Argument-validation tests completed.
- [ ] Permission-denial tests completed.
- [ ] Prompt-injection tests completed.
- [ ] Error-recovery tests completed.
- [ ] Cost and latency reviewed.
- [ ] Auditability verified.

## Operations

- [ ] Monitoring dashboard active.
- [ ] Incident response path defined.
- [ ] Human escalation route defined.
- [ ] Review cadence scheduled.
