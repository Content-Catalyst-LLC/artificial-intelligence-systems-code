# AI Agents, Tool Use, and Workflow Automation

This article folder supports the Artificial Intelligence Systems knowledge-series article:

AI Agents, Tool Use, and Workflow Automation

It includes:

- Python workflow for agent tool-use evaluation, workflow scoring, permission review, prompt-injection exposure review, and governance summaries.
- R workflow for workflow automation evaluation summaries by workflow type and tool risk.
- SQL schema for agent registries, tool catalogs, workflow runs, tool-call logs, permission checks, human approvals, sandbox events, incidents, and governance reviews.
- Rust CLI for validating agent workflow records and safety thresholds.
- Go monitoring API for agent governance metrics.
- Julia workflow for agent step-risk, tool-risk, and workflow utility simulation.
- TypeScript dashboard logic for agent governance review.
- C++ example for tool-risk scoring and permission-gated execution.
- Advanced notebooks for tool-call evaluation, workflow traces, permission review, prompt-injection testing, rollback analysis, and governance registers.
- Documentation templates for agent system cards, tool registry review, permission review, sandbox review, human approval review, workflow incident response, and agent deployment readiness.
