# Probabilistic Machine Learning and Bayesian AI Systems

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Probabilistic Machine Learning and Bayesian AI Systems

It includes:

- Python workflow for Bayesian updating, posterior risk estimates, calibration review, and expected-loss decision support.
- R workflow for probabilistic forecast evaluation, calibration bins, Brier score, and subgroup review.
- SQL schema for Bayesian model records, priors, likelihoods, posterior summaries, prediction logs, calibration audits, decision thresholds, and governance review.
- Rust CLI for validating probabilistic forecast records and calibration thresholds.
- Go monitoring API for probabilistic AI governance metrics.
- Julia workflow for Bayesian sequential updating and uncertainty propagation.
- TypeScript dashboard logic for probabilistic risk and calibration review.
- C++ example for posterior updating and expected-loss prioritization.
- Advanced notebooks for Bayesian inference, calibration, Gaussian-process intuition, approximate inference diagnostics, and governance registers.
- Documentation templates for Bayesian model cards, prior review, calibration review, uncertainty communication, and decision-threshold approval.
