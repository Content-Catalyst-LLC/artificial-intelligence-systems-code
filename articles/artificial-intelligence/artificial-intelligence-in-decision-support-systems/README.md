# Artificial Intelligence in Decision Support Systems

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Artificial Intelligence in Decision Support Systems

It includes:

- Python workflow for decision scoring, uncertainty, robust utility, and governance summaries.
- R workflow for scenario review and decision-quality analysis.
- SQL schema for decision systems, options, predictions, utilities, scenarios, recommendations, overrides, audits, and governance review.
- Rust CLI for validating decision records and risk thresholds.
- Go monitoring API for decision-support system health and governance metrics.
- Julia workflow for robust decision simulation and expected-utility optimization.
- TypeScript dashboard logic for human-facing decision review.
- C++ example for high-performance action ranking and utility scoring.
- Advanced notebooks for expected utility, causal decision support, robust optimization, and governance.
- Documentation templates for decision review, human override, and institutional accountability.
