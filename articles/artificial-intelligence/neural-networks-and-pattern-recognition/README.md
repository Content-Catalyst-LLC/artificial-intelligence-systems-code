# Neural Networks and Pattern Recognition

This article folder supports the article "Neural Networks and Pattern Recognition" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- multilayer perceptron classification
- nonlinear pattern recognition
- representation geometry
- activation function intuition
- backpropagation concepts
- generalization diagnostics
- grouped neural-network error analysis
- SQL metadata for neural-network systems
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
