# Education AI Deployment Readiness Checklist

## Purpose and Policy

- [ ] Intended use documented.
- [ ] Prohibited use documented.
- [ ] AI-use disclosure policy defined.
- [ ] Assessment implications reviewed.
- [ ] Teacher or supervisor oversight defined.

## Learning Evaluation

- [ ] Learning gain evaluation defined.
- [ ] Independent transfer evaluation defined.
- [ ] Feedback quality reviewed.
- [ ] Assistance-gap monitoring defined.
- [ ] Dependency risk reviewed.

## Privacy, Accessibility, and Equity

- [ ] Student or worker data reviewed.
- [ ] Vendor data policy reviewed.
- [ ] Accessibility review completed.
- [ ] Bias and equity review completed.
- [ ] Incident response route defined.

## Operations

- [ ] Monitoring signals active.
- [ ] Change-control process defined.
- [ ] Training materials prepared.
- [ ] Governance review scheduled.
