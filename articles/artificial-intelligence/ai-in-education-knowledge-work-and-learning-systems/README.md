# AI in Education, Knowledge Work, and Learning Systems

This article folder supports the Artificial Intelligence Systems knowledge-series article:

AI in Education, Knowledge Work, and Learning Systems

It includes:

- Python workflow for learning gain, independent transfer, AI assistance gaps, feedback quality, dependency risk, privacy risk, assessment substitution risk, and governance review.
- R workflow for AI learning system evaluation summaries.
- SQL schema for education AI systems, intended use, learning events, assessment evidence, AI-use disclosure, privacy review, accessibility review, subgroup review, incidents, change control, and governance review.
- Rust CLI for validating learning-system monitoring records and governance thresholds.
- Go monitoring API for education AI and learning-system metrics.
- Julia workflow for learning gain and assistance-gap simulation.
- TypeScript dashboard logic for learning-system governance review.
- C++ example for learning-support routing and governance-risk scoring.
- Advanced notebooks for learning gain, assessment validity, feedback quality, assistance gaps, equity review, privacy review, and governance registers.
- Documentation templates for education AI system cards, AI-use policy, assessment redesign, privacy review, accessibility review, bias and equity review, incident response, and deployment readiness.

This repository is for educational systems analysis and governance workflows. It does not make student-specific decisions, grade students, determine academic integrity violations, or provide legal advice.
