# Retrieval-Augmented Generation and AI Knowledge Systems

This article folder supports the Artificial Intelligence Systems knowledge-series article:

Retrieval-Augmented Generation and AI Knowledge Systems

It includes:

- Python workflow for RAG evaluation, retrieval quality scoring, answer grounding, citation fidelity, security review, and governance summaries.
- R workflow for query-type and source-type RAG evaluation review.
- SQL schema for source registries, document chunks, embeddings, retrieval logs, reranking records, prompt versions, citations, access controls, evaluation results, and governance reviews.
- Rust CLI for validating RAG evaluation records and source-support thresholds.
- Go monitoring API for RAG knowledge-system governance metrics.
- Julia workflow for retrieval recall, grounding, and citation-fidelity simulation.
- TypeScript dashboard logic for RAG governance review.
- C++ example for hybrid retrieval scoring and context-budget selection.
- Advanced notebooks for chunking diagnostics, retrieval evaluation, citation support, prompt-injection testing, access-control review, and governance registers.
- Documentation templates for RAG system cards, source registry review, chunking review, retrieval evaluation, citation audit, and prompt-injection incident response.
