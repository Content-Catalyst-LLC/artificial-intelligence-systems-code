# Dataset Documentation Template

Recommended fields:

- dataset name
- dataset version
- dataset owner
- intended use
- prohibited or inappropriate uses
- source systems
- collection period
- target population
- represented populations
- underrepresented or missing populations
- label source
- labeling procedure
- measurement constructs
- proxy variables
- known measurement limitations
- missingness patterns
- bias audit findings
- quality thresholds
- governance owner
- review schedule
