# Data Quality, Bias, and Measurement in Machine Learning

This article folder supports the article "Data Quality, Bias, and Measurement in Machine Learning" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- data-quality diagnostics
- missingness analysis
- subgroup representation audits
- measurement-error simulation
- label-noise analysis
- fairness metric calculation
- bias taxonomy documentation
- SQL schemas for data quality and bias governance
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
