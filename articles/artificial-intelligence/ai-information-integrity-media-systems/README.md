# AI, Information Integrity, and Media Systems

This repository folder supports the Sustainable Catalyst article:

**AI, Information Integrity, and Media Systems**

It provides reproducible examples for information-integrity risk scoring, source-diversity monitoring, provenance coverage, correction-effectiveness analysis, media-system governance, and public-accountability workflows.

## Contents

- `python/information_integrity_risk_scoring.py` — content-level information-integrity risk scoring
- `r/source_diversity_media_monitoring.R` — source diversity, provenance coverage, and correction-effectiveness workflow
- `sql/media_integrity_governance_schema.sql` — governance schema for provenance, sources, corrections, amplification, and monitoring
- `rust/` — command-line information-risk checker
- `go/` — lightweight media-integrity monitoring service
- `julia/source_diversity_sensitivity.jl` — source-diversity sensitivity analysis
- `typescript/` — media-integrity governance record validator
- `cpp/` — high-throughput information-risk scoring example
- `docs/templates/` — provenance review, editorial AI disclosure, correction tracking, and information-integrity audit templates
- `notebooks/` — notebook placeholder for expanded analysis
- `scripts/run_all.sh` — runs available workflows
