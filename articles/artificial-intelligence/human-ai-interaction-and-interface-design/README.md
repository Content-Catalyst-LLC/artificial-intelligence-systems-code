# Human–AI Interaction and Interface Design

This article folder supports the article "Human–AI Interaction and Interface Design" in the Artificial Intelligence Systems knowledge series.

It includes reproducible examples for:

- user reliance diagnostics
- overreliance and underreliance analysis
- interface-risk modeling
- explanation and uncertainty display diagnostics
- scenario-based human-AI evaluation
- SQL metadata for interaction logs and design reviews
- governance checklists
- advanced Jupyter notebooks

The examples are educational and synthetic unless otherwise documented.
